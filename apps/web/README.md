# Spaila Web

Phase 1 web scaffold for the operational mirror.

The web app is intentionally scoped to shared operational workflows:

- Active and completed orders
- Order detail
- Conversations
- Attachments
- Archive search
- Settings
- Account, billing, and support foundations

Desktop remains authoritative for parser, inbox ingestion, helper control, backup/restore, and local filesystem operations.

Build with:

```powershell
npm run build:web
```
